# Geo SEO / International Readiness Checklist

## 1) URL structure
- Use country/language routes consistently (`/us/`, `/uk/`, `/fr/`, `/ma-fr/`, `/ar/`, `/ma-ar/`).
- Keep one canonical per page variant.

## 2) hreflang
- Add hreflang tags in page `<head>`.
- Mirror hreflang alternates in sitemap.
- Always include `x-default`.

## 3) Localization
- Translate title, meta description, H1, CTA, and FAQ per locale.
- Adapt examples, currency, and wording to region.

## 4) Technical
- Submit sitemap to Google Search Console.
- Set geo targets where appropriate (if using ccTLD/subdomains).
- Ensure fast performance globally (CDN + compressed assets).

## 5) Conversion
- Show locale switcher.
- Default locale from browser language, but allow manual override.
- Keep forms and legal text localized.

## 6) Analytics
- Track conversions by locale and country.
- Compare opt-in and paid waitlist rates per market.
