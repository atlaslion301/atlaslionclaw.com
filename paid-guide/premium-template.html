1) Beginner Track (Non-Technical How-To)

Use this first if you're setting up OpenClaw for the first time and want plain-English steps.
  1. Install OpenClaw and verify with openclaw gateway status.
  2. Add one channel token (Discord or Telegram) first.
  3. Set allowlist to your own user ID.
  4. Restart gateway and send test message.
  5. If no response, use the troubleshooting matrix in Section 4.

Success check: OpenClaw replies consistently in your test channel.
